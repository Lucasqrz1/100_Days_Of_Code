# Python Flask and Decorators Examples

This repository contains examples of Python Flask web application and decorator patterns.

## Files Overview

### hello.py

A simple Flask web application that:

- Creates a basic web server
- Serves "Hello World!" at the root endpoint
- Includes examples of custom decorators
- Demonstrates basic Flask routing

### exercise.py

A demonstration of practical decorator usage that:

- Implements a speed calculation decorator
- Shows how to measure function execution time
- Includes examples of fast and slow functions
- Demonstrates decorator usage with different complexity levels
