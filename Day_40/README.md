# Flight Deals Tracker

A Python application that tracks flight prices and notifies users about deals via email and WhatsApp messages. The application integrates with external flight search APIs and Google Sheets to manage destination data and user preferences.

## Features

- 🔍 Automated flight price tracking
- ✈️ Support for multiple destinations
- 📧 Email notifications for price drops
- 📱 WhatsApp notifications
- 📊 Google Sheets integration for data management
- 🕒 Configurable search period
- ✈️ Direct and multi-stop flight support

## Prerequisites

- Python 3.13.0 or higher
- virtualenv (for environment management)
- Google Sheets API access
- Flight search API credentials
- WhatsApp Business API credentials
