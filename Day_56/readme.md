# Personal Portfolio Website

A minimalist personal portfolio website built using the Identity template from HTML5 UP. This responsive single-page website is perfect for creating a simple yet elegant online presence.

![Website Preview](static/images/preview.png)

## 🚀 Features

- Clean, responsive single-page design
- Profile section with avatar
- Social media integration
- Mobile-friendly layout
- Cross-browser compatibility
- Easy to customize and maintain

## 💻 Technologies Used

- HTML5
- CSS3
- JavaScript
- [HTML5 UP Identity Template](https://html5up.net/identity)
