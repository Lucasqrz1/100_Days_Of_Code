# ☕ OOP Coffee Machine – Day 16

This is a small project for Day 16 of my **100 Days of Code** challenge. It’s a console-based coffee machine simulator built using **Object-Oriented Programming (OOP)** in Python.

## 💡 What it does

- Lets you choose a drink: `espresso`, `latte`, or `cappuccino`
- Checks if there are enough ingredients
- Takes coins and gives change if needed
- Shows a report of ingredients and money
- Lets you turn off the machine with the command `off`

## 🗂️ Files

- `main.py` – Runs the program
- `menu.py` – Handles the coffee menu
- `coffee_maker.py` – Manages water, milk, and coffee
- `money_machine.py` – Manages payments and money
