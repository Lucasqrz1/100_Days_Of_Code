# 100 Days of Code

This repository contains my progress through the #100DaysOfCode challenge, based on "The Complete Python Pro Bootcamp" by Dr. Angela Yu.

## Overview

- **Duration:** 100 consecutive days
- **Commitment:** Minimum of 1 hour of coding per day
- **Objective:** Strengthen Python skills through daily practice and project development

## Contents

This repository includes a variety of Python projects developed throughout the challenge, such as:

- Blackjack game
- Calculator
- Snake game
- Pong game
- Caesar cipher
- Password generator
- Mail merge tool
- Number guessing game
- Hangman
- Turtle race

Each day’s work is organized into folders with clear file names and comments.
